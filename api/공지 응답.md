## 공지 응답

| 메소드 | 경로                  | 짧은 설명 |
| ------ | --------------------- | --------- |
| POST   | /info/response/notice | 공지 응답 |

### 요청 헤더

```json
Content-Type: application/json,
Authorization: token
```

### 요청 바디

```json
{
    "notice_idx" : 1
}
```

### 응답 바디

#### 컨텐츠 작성 성공

```json
{
    "status" : 201,
    "message" : "Success Response Notice"
}
```

#### 값이 없을 경우

```json
{
    "status": 400,
    "message": "Null Value"
}
```

#### token값에 문제가 있을 경우

```json
{
    "status": 400,
    "message": "Verification Failed"
}
```

#### 서버 내부 에러

```json
{
    "status": 500,
    "message": "Internal Server Error"
}
```
