## 회원가입

| 메소드 | 경로           | 짧은 설명 |
| ------ | -------------- | --------- |
| POST   | /auth/register | 회원가입  |

### 요청 헤더

```json
Content-Type: application/json
```

### 요청 바디

```json
{
    "status" : 201,
    "id" : "dd@gmail.com",
    "pwd" : "dd",
    "name" : "gd",
	"phone" : "010-1234-5678"
}
```

### 응답 바디

#### 컨텐츠 작성 성공

```json
{
    "status": 201,
    "message": "Success Register"
}
```

#### 값이 없을 경우

```json
{
    "status": 400,
    "message": "Null Value"
}
```

#### 아이디 중복일 경우

```json
{
    "status": 400,
    "message": "ID Already Exist"
}
```

#### 서버 내부 에러

```json
{
    "status": 500,
    "message": "Internal Server Error"
}
```
